# Plants-Vs-Zombies-Recreation
A console-based Plants vs. Zombies-style strategy game built in C++ using object-oriented programming, inheritance, polymorphism, and dynamic memory management. Features grid-based gameplay, plant placement, zombie movement, combat, and health tracking.
